## üîç Discover ‚Äî Detail (25 minutes)

### Part 1 ‚Äî Common Factors

{{% notice style="discover" title="Key idea" %}}
Find the **HCF** of all terms, then divide each term by it.

$$6x^2 + 9x = 3x(2x + 3)$$
{{% /notice %}}

**I Do:** Factorise $12x^3 - 8x^2 + 4x$

| Step | Working |
|---|---|
| Find HCF | HCF $= 4x$ |
| Divide each term | $3x^2,\; -2x,\; 1$ |
| Write answer | $4x(3x^2 - 2x + 1)$ |

**We Do:** a) $5x^2 - 15x$ ‚Üí $5x(x-3)$ &nbsp;&nbsp; b) $6x^3 + 9x^2 - 3x$ ‚Üí $3x(2x^2+3x-1)$

---

### Part 2 ‚Äî Monic Quadratics

{{% notice style="discover" title="Key idea" %}}
For $x^2 + bx + c$, find $p$ and $q$ with $pq = c$ **and** $p + q = b$:

$$x^2 + bx + c = (x + p)(x + q)$$
{{% /notice %}}

**I Do:** Factorise $x^2 - 3x - 28$

| Step | Working |
|---|---|
| Need $pq = -28$, $p+q = -3$ | Factor pairs of $-28$: ‚Ä¶ $(4, -7)$ ‚Ä¶ |
| Which pair sums to $-3$? | $4 + (-7) = -3$ ‚úì |
| Write answer | $(x + 4)(x - 7)$ |

**We Do:** a) $x^2 + 9x + 20$ ‚Üí $(x+4)(x+5)$ &nbsp;&nbsp; b) $x^2 - 2x - 15$ ‚Üí $(x-5)(x+3)$

---

### Part 3 ‚Äî Non-Monic Quadratics (AC Method)

{{% notice style="discover" title="Key idea ‚Äî AC method" %}}
For $ax^2 + bx + c$:
1. Find $ac$
2. Find $p, q$ with $pq = ac$ and $p + q = b$
3. Split the middle term: $ax^2 + px + qx + c$
4. Factor by grouping
{{% /notice %}}

**I Do:** Factorise $6x^2 + 11x - 10$

| Step | Working |
|---|---|
| $ac = 6 \times (-10)$ | $= -60$ |
| Find pair: $pq=-60$, $p+q=11$ | $15 \times (-4) = -60$; $15 + (-4) = 11$ ‚úì |
| Split | $6x^2 + 15x - 4x - 10$ |
| Group | $3x(2x+5) - 2(2x+5)$ |
| Answer | $(3x-2)(2x+5)$ |

**We Do:** a) $2x^2 + 7x + 3$ ‚Üí $(2x+1)(x+3)$ &nbsp;&nbsp; b) $3x^2 - 10x + 8$ ‚Üí $(3x-4)(x-2)$

---

### Part 4 ‚Äî Special Identities

{{% notice style="discover" title="Two identities to recognise" %}}
**Perfect square:** $a^2 \pm 2ab + b^2 = (a \pm b)^2$

**Difference of two squares:** $a^2 - b^2 = (a+b)(a-b)$

These are the reverse of the special expansions from 3A ‚Äî recognising the pattern is the skill.
{{% /notice %}}

**I Do:** $9x^2 - 24x + 16 = (3x-4)^2$ &nbsp;&nbsp;&nbsp; $25x^2 - 36 = (5x+6)(5x-6)$

**We Do:** a) $x^2 - 16x + 64$ ‚Üí $(x-8)^2$ &nbsp;&nbsp; b) $16x^2 - 81$ ‚Üí $(4x+9)(4x-9)$
