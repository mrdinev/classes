$$a(b + c) = ab + ac$$

### Part 3 â€” Special Expansions (5 min)

{{% notice style="discover" title="Two useful identities" %}}
**Perfect square:** $(a + b)^2 = a^2 + 2ab + b^2$
- Example: $(x - 4)^2 = x^2 - 8x + 16$

**Difference of two squares:** $(a + b)(a - b) = a^2 - b^2$
- Example: $(x - 3)(x + 3) = x^2 - 9$
